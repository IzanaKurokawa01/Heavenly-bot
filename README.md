# Heavenly Dynasty Telegram Bot

Telegram бот для отслеживания статистики клубов Brawl Stars.

## Команды:
- `/start` - Главное меню
- `/rating` - Рейтинг клубов
- `/update` - Обновить данные через API
- `/search` - Поиск клуба
- `/help` - Помощь
- `/status` - Статус бота

## Хостинг:
- Render.com (бесплатный тариф)
- Cloudflare Proxy для API Brawl Stars

## Технологии:
- Python 3.11
- python-telegram-bot
- Brawl Stars API
